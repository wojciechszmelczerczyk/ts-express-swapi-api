# Get favorite film lists

## Description

Get all favorite film lists.

<b>URL :</b> `/favorites`

<b>Method:</b> `GET`

## Success Response

Code: `200 OK`

Return id and name of all lists.

### Context example

```json
[
  {
    "id": 1,
    "name": "Dark Saga"
  },
  {
    "id": 2,
    "name": "Light Saga"
  }
]
```
